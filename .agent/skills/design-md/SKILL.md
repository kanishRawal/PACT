# design-md Skill

When invoked, act as a UI/UX expert to generate a comprehensive `DESIGN.md` file for the project. Analyze any provided requirements or existing Stitch screens.

## Responsibilities:
1. Define the core color palette (Primary, Secondary, Accent, Background, Text) with exact HSL/Hex values.
2. Outline typography rules (Font Family, Headings, Body, Monospace).
3. Specify spacing systems, border radii, and elevation (shadows/depth).
4. Outline specific micro-interactions and animations (e.g., hover states, transition durations).
5. If using `StitchMCP`, base this design system on the generated project vibe.
